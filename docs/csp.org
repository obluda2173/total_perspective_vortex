* Supplementary

** Variance

Variance quantifies how far a set of numbers is spread out from its mean.

$$s^2 = \mathrm{var}(x) = \dfrac{\sum_{i=1}^{n} (x_i - \bar{x})^2}{n-1}$$

where:
- $s^2$ :: the sample variance
- $n$ :: the number of observations in the sample
- $x_i$ :: the $i$ th element of the sample
- $\bar{x}$ :: the sample mean

Facts:
- The variance is expressed in squared units of the original data.
- Because deviations from the mean are squared, values unusually far from the mean (outliers) have a disproportionately large impact.
- The standard deviation is the square root of the variance ($\sigma$ or $s$).

** Covariance

Covariance measures how two variables change together:

$$\mathrm{cov}(x, y) = \dfrac{\sum_{i=1}^{n} (x_i - \bar{x}) (y_i - \bar{y})}{n-1}$$

** Covariance matrix

\begin{align*}
\Sigma &=
\begin{bmatrix}
\mathrm{var}(x) & \mathrm{cov}(x, y) \\
\mathrm{cov}(y, x) & \mathrm{var}(y)
\end{bmatrix}
\end{align*}

If the covariance matrix takes the form of:

- *Sphere* (equal variance):

  \begin{align*}
  \Sigma &=
  \begin{bmatrix}
  1 & 0 \\
  0 & 1
  \end{bmatrix}
  \end{align*}

  $x$ and $y$ have no linear relationship.

- *Axis-aligned ellipse* (unequal variance):

  \begin{align*}
  \Sigma &=
  \begin{bmatrix}
  3 & 0 \\
  0 & 1
  \end{bmatrix}
  \end{align*}

  The data stretches along the horizontal axis but stays tight vertically. $x$ and $y$ still have no linear relationship.

- *Tilted ellipse* (non-zero covariance):

  \begin{align*}
  \Sigma &=
  \begin{bmatrix}
  2 & 1.5 \\
  1.5 & 2
  \end{bmatrix}
  \end{align*}

  The positive covariance ($1.5$) tilts the ellipse from bottom-left to top-right: as $x$ increases, $y$ tends to increase.

** Common Average Reference

Common Average Reference (CAR) is a spatial filter used in EEG recordings.

$$V_{\mathrm{CAR}, i} = V_i - \dfrac{1}{N} \sum_{j=1}^{N} V_j$$

where:

- $V_{\mathrm{CAR}}$ :: is the new re-referenced voltage of electrode

- $V_i$ :: is the original electrode redording

- $N$ :: is the total number of electrodes

- $\sum_{j=1}^{N} V_j$ :: is the sum of voltage across all electrodes

** Singular matrix

Let $A$ be an $n \times n$ matrix over field $F$.

The matrix $A$ is called *singular* if it has not invertable. Such that,

$$\neg \exists B \in F^{n \times n} : AB = BA = I$$

Key concepts:

- *Determinant*: $\mathrm{det}(A) = 0$

- *Kernel*: $Ax = 0$ has non-trivial solution

- *Deficient rank*: $\mathrm{rank}(A) < n$

- *Zero eigenvalue*: The scalar $0$ is an eigenvalue of $A$

** Properties used later

For any $\Sigma^{(c)}$ built as $ZZ^{\top}$ (up to a positive scalar):

1. *Symmetry.* $(ZZ^{\top})^{\top} = ZZ^{\top}$.
2. *Positive semi-definiteness.* $v^{\top}ZZ^{\top}v = \lVert Z^{\top}v \rVert_2^2 \ge 0$ for all $v$.
3. *Definiteness.* $\Sigma^{(c)} \succ 0$ (strictly) iff $Z$ has full row rank $C$. If the data carries an average-reference constraint $\sum_{\text{channels}} = 0$, then $\mathbf{1}^{\top}Z = 0$, so $\operatorname{rank} Z \le C-1$ and $\Sigma^{(c)}$ is *singular*.

Property 3 is the whole reason for section [[*Rank restriction][Rank restriction]].

** Definitions

- Epoch :: a time interval of one EEG activity (equivalently: one trial).

- Class :: a type of EEG activity (hands movement, feet movement, ...).

- Matrix pencil :: the one-parameter family $A - \lambda B$ generated by a pair of matrices $(A, B)$. Solving the pencil means solving the *generalised eigenproblem* $Aw = \lambda Bw$. For $B = I$ this reduces to the ordinary eigenproblem.

- Generalised Rayleigh quotient :: for symmetric $A$ and symmetric positive definite $B$, the scalar field $J(w) = \dfrac{w^{\top}Aw}{w^{\top}Bw}$, defined for $w \neq 0$.

** Symbols

- $C$ :: number of EEG channels

- $T$ :: number of time samples in one epoch

- $N$ :: total number of epochs

- $X_i$ :: epoch $i$, a $C \times T$ matrix (channel $\times$ time), band-pass filtered

- $y_i$ :: class label of epoch $i$

- $\mathcal{J}_{c}$ :: index set of epochs in class $c$, $\; c \in \{+, -\}$

- $N_c = |\mathcal{J}_{c}|$ :: number of epochs in class $c$

- $Z_c$ :: class-$c$ epochs concatenated along time, a $C \times N_cT$ matrix

- $\Sigma^{(c)}$ :: spatial covariance of class $c$, a $C \times C$ matrix

- $\Sigma = \Sigma^{(+)} + \Sigma^{(-)}$ :: composite covariance

- $w$ :: one spatial filter, a $C$ -vector

- $W = [\,w_1 \; \cdots \; w_C\,]$ :: filter matrix, $C \times C$, *columns* are filters

- $\Lambda^{(c)}$ :: diagonal matrix of filtered variances for class $c$

- $\lambda$ :: eigenvalue of the pencil $(\Sigma^{(+)}, \Sigma^{(-)})$

- $\mu$ :: eigenvalue of the pencil $(\Sigma^{(+)}, \Sigma)$

*Convention warning.* The subject writes $W^{\top}X = X_{CSP}$, so $W$ has filters in its *columns* and this document follows that. Both MNE and my code store =filters_= $= W^{\top}$, i.e. filters in *rows*. Every shape assertion in the test harness is against the row convention.

* Objective

Find a spatial filter $w \in \mathbb{R}^{C}$ whose projected signal $w^{\top}X_i$ carries large variance for one class and small variance for the other. Formally, maximise the generalised Rayleigh quotient

#+NAME: eq:objective
\begin{equation}
J(w) = \frac{w^{\top}\Sigma^{(+)}w}{w^{\top}\Sigma^{(-)}w}
\end{equation}

The numerator is the class-$(+)$ variance of the projected signal, the denominator the class-$(-)$ variance (see [[*Variance of a projection][Variance of a projection]]). $J$ is *scale invariant*: $J(\alpha w) = J(w)$ for $\alpha \neq 0$, so a filter is only ever determined up to a non-zero scalar. Maximising $J$ gives the most $( +)$ -dominant filter; minimising it gives the most $(-)$ -dominant one.

** DONE Bridge lemma :: stationarity $\Rightarrow$ generalised eigenproblem

The step the rest of the document depends on, and the one I derive myself:

#+BEGIN_QUOTE
*Lemma.* $w$ is a stationary point of $J$ if and only if

$$\Sigma^{(+)}w = \lambda\,\Sigma^{(-)}w \text{ with } \lambda = J(w)$$
#+END_QUOTE

*Proof.*

\begin{align*}
J(w) &= \dfrac{w^{\top}\Sigma^{(+ )}w}{w^{\top}\Sigma^{(-)}w} \\
J(w) &= \dfrac{u}{v} \qquad \left( u = w^{\top}\Sigma^{(+)}w,\ v = w^{\top}\Sigma^{(-)}w \right) \\
\nabla J(w) &= \dfrac{\nabla u v - u \nabla v}{v^{2}} \\
\nabla u &= \Sigma^{(+)} w \\
\nabla v &= \Sigma^{(-)} w \\
\nabla J(w) &= \dfrac{(\Sigma^{(+ )} w) (w^{\top}\Sigma^{(-)}w) - (w^{\top}\Sigma^{(+)}w) (\Sigma^{(-)} w)}{(w^{\top}\Sigma^{(-)}w)^{2}} \\
\end{align*}

\begin{align*}
0 &= \dfrac{(\Sigma^{(+ )} w) (w^{\top}\Sigma^{(-)}w) - (w^{\top}\Sigma^{(+)}w) (\Sigma^{(-)} w)}{(w^{\top}\Sigma^{(-)}w)^{2}} \\
\implies 0 &= (\Sigma^{(+ )} w) (w^{\top}\Sigma^{(-)}w) - (w^{\top}\Sigma^{(+)}w) (\Sigma^{(-)} w) \\
\end{align*}

\begin{align*}
(\Sigma^{(+ )} w) (w^{\top}\Sigma^{(-)}w) &= (w^{\top}\Sigma^{(+ )}w) (\Sigma^{(-)} w) \\
(\Sigma^{(+ )} w) &= \dfrac{(w^{\top}\Sigma^{(+ )}w)}{(w^{\top}\Sigma^{(-)}w)} (\Sigma^{(-)} w) \\
(\Sigma^{(+ )} w) &= J(w) (\Sigma^{(-)} w) \\
\Sigma^{(+ )} w &= \lambda \Sigma^{(-)} w \qquad (\lambda = J(w)) \qed \\
\end{align*}

* Estimation of the covariances

** Zero-mean assumption

$\Sigma^{(c)}$ is written below as an uncentred second moment $\frac{1}{n-1}ZZ^{\top}$ rather than a centred covariance $\frac{1}{n-1}(Z-\bar{z})(Z-\bar{z})^{\top}$. This is legitimate *because* the signal was band-pass filtered to $[7, 30]$ Hz beforehand: the filter has no DC gain, so $\bar{z} \approx 0$ per channel and the two estimators coincide to within the filter's stopband attenuation. The preprocessing choice and the estimator choice are therefore coupled — dropping the high-pass would invalidate this.

** Concatenation estimator (=cov_est='concat'=)

Concatenate all class-$c$ epochs along the time axis:

#+NAME: eq:concat
\begin{equation}
Z_c = \big[\, X_{i_1} \;\; X_{i_2} \;\; \cdots \;\; X_{i_{N_c}} \,\big]
\in \mathbb{R}^{C \times N_c T},
\qquad \{i_1,\dots,i_{N_c}\} = \mathcal{J}_c
\end{equation}

#+NAME: eq:sigma
\begin{equation}
\Sigma^{(c)} = \frac{Z_c Z_c^{\top}}{N_cT - 1}
\end{equation}

In code:

#+BEGIN_SRC python
Z = X.transpose(1, 0, 2).reshape(n_channels, -1)   # Z_c
return Z @ Z.T / (Z.shape[1] - 1)                  # Sigma^(c)
#+END_SRC

The transpose reorders $(N_c, C, T) \to (C, N_c, T)$ and the reshape flattens the last two axes in C-order, which is exactly the horizontal concatenation of eq. [[eq:concat]].

** Relation to the per-epoch estimator (=cov_est='epoch'=)

Since $Z_cZ_c^{\top} = \sum_{i \in \mathcal{J}_c} X_iX_i^{\top}$, the two common estimators differ *only by a positive scalar*:

\begin{align*}
\Sigma^{(c)}_{\text{concat}} &= \frac{1}{N_cT-1}\sum_{i \in \mathcal{J}_c} X_iX_i^{\top} \\
\Sigma^{(c)}_{\text{epoch}}  &= \frac{1}{N_c}\sum_{i \in \mathcal{J}_c} \frac{X_iX_i^{\top}}{T-1}
\end{align*}

so $\Sigma^{(c)}_{\text{concat}} = \kappa_c \, \Sigma^{(c)}_{\text{epoch}}$ with

\begin{align*}
\kappa_c &= \dfrac{\dfrac{1}{N_cT-1}\sum_{i \in \mathcal{J}_c} X_iX_i^{\top}}{\dfrac{1}{N_c}\sum_{i \in \mathcal{J}_c} \dfrac{X_iX_i^{\top}}{T-1}} \\
\kappa_c &= \dfrac{\dfrac{1}{N_cT-1}\sum_{i \in \mathcal{J}_c} X_iX_i^{\top}}{\dfrac{1}{N_c} \dfrac{1}{T-1} \sum_{i \in \mathcal{J}_c}X_iX_i^{\top}} \\
\kappa_c &= \dfrac{\dfrac{1}{N_cT-1}}{\dfrac{1}{N_cT-N_c}} = \dfrac{N_cT-N_c}{N_cT-1} \\
\kappa_c &= \frac{N_c(T-1)}{N_cT-1} = \frac{T-1}{T - 1/N_c}
\end{align*}

A *common* positive scalar on both classes cancels in eq. [[eq:objective]] entirely, so the estimators would be equivalent if $\kappa_+ = \kappa_-$. They are not: $\kappa_c$ depends on $N_c$, and

$$\kappa_+ - \kappa_- \;\approx\; \frac{1}{T}\left(\frac{1}{N_-} - \frac{1}{N_+}\right)$$

which is non-zero under class imbalance. At $N_+ = 22$, $N_- = 23$, $T = 161$ this is $1.2 \times 10^{-5}$ relative, and it displaces the spectrum by

$$\max_k \big|\mu_k^{\text{concat}} - \mu_k^{\text{epoch}}\big| = 3.07 \times 10^{-6}$$

measured. That is *above* the harness's =EVAL_ATOL = 1e-6=. So the two estimators are interchangeable for classification but not for bit-level parity with MNE: the estimator choice must match =cov_est=, and =cov_est='concat'= is the MNE default. The estimators diverge further under regularisation, which is applied once per class in one case and once per epoch in the other.

* The pencil

** Unnormalised form

From the bridge lemma:

#+NAME: eq:pencil-raw
\begin{equation}
\Sigma^{(+)} w = \lambda \, \Sigma^{(-)} w
\end{equation}

The textbook next step,

\begin{align*}
(\Sigma^{(-)})^{-1}\Sigma^{(+)} w &= \lambda w \\
(M - \lambda I) w &= 0 \qquad \big(M = (\Sigma^{(-)})^{-1}\Sigma^{(+)}\big) \\
&\Rightarrow \det(M - \lambda I) = 0
\end{align*}

is *correct but unusable*, for three reasons:

1. It requires $\Sigma^{(-)} \succ 0$. Under an average reference this fails (Supplementary, property 3).
2. $M$ is a product of two symmetric matrices and is *not itself symmetric*, so the spectrum is not guaranteed real and the eigenvectors are not guaranteed orthogonal in any inner product. Numerically this forces a general (non-symmetric) solver, i.e. =numpy.linalg.eig=, whose return dtype is =complex128= as soon as any imaginary part is non-zero.
3. $\det(M - \lambda I) = 0$ is a characteristic-polynomial statement. It defines the eigenvalues but is never how they are computed for $C = 64$.

** Normalised form

Add $\lambda \Sigma^{(+)}w$ to both sides of eq. [[eq:pencil-raw]]:

\begin{align*}
\Sigma^{(+)}w + \lambda\Sigma^{(+)}w &= \lambda\Sigma^{(-)}w + \lambda\Sigma^{(+)}w \\
(1+\lambda)\,\Sigma^{(+)}w &= \lambda\,\big(\Sigma^{(+)} + \Sigma^{(-)}\big)w \\
\Sigma^{(+)}w &= \frac{\lambda}{1+\lambda}\,\Sigma w
\end{align*}

#+NAME: eq:pencil-norm
\begin{equation}
\Sigma^{(+)} w = \mu \, \Sigma w,
\qquad \mu = \frac{\lambda}{1+\lambda},
\qquad \lambda = \frac{\mu}{1-\mu}
\end{equation}

*The eigenvectors are unchanged.* Only the eigenvalue is reparametrised, by a monotone bijection $[0,\infty) \to [0,1)$. The two pencils therefore produce identical spatial filters and identical features. What changes:

1. *Interpretation.* By the Rayleigh quotient,

   $\mu = \dfrac{w^{\top}\Sigma^{(+)}w}{w^{\top}\Sigma^{(+)}w + w^{\top}\Sigma^{(-)}w}$

   — the fraction of the filter's output power belonging to class $( +)$. $\mu \to 1$: pure $( +)$. $\mu \to 0$: pure $(-)$. $\mu = \tfrac12$: non-discriminative. This gives a *testable invariant*, $\mu \in [0,1]$, and the correct ordering criterion $|\mu - \tfrac12|$.

2. *Conditioning.* $\Sigma = \Sigma^{(+)} + \Sigma^{(-)}$ is singular only in directions where *both* classes are silent, a strictly weaker condition than $\Sigma^{(-)}$ being singular.

3. *Symmetry.* $\Sigma$ is symmetric PSD, so if it is also definite the pencil $(\Sigma^{(+)}, \Sigma)$ is a *symmetric-definite* pencil and the machinery of the next section applies.

** Complementarity

Applying the same algebra with the classes swapped gives $\mu^{(-)} = 1 - \mu$ for the same $w$. Collected over all filters this is the identity

#+NAME: eq:complement
\begin{equation}
\Lambda^{(+)} + \Lambda^{(-)} = I
\end{equation}

which is *not* free: it holds only under the normalisation of the next section.

* Simultaneous diagonalisation

** Statement

Choose the columns of $W$ to be the generalised eigenvectors of eq. [[eq:pencil-norm]], scaled so that

#+NAME: eq:normalisation
\begin{equation}
W^{\top}\Sigma W = I
\end{equation}

Then

#+NAME: eq:diag
\begin{equation}
W^{\top}\Sigma^{(+)}W = \Lambda^{(+)} = \operatorname{diag}(\mu_1,\dots,\mu_C),
\qquad
W^{\top}\Sigma^{(-)}W = \Lambda^{(-)} = I - \Lambda^{(+)}
\end{equation}

Eq. [[eq:diag]] is the actual content of "CSP": a single change of basis that diagonalises *both* class covariances at once, with the two diagonals summing to the identity. Eq. [[eq:complement]] is then immediate from $\Sigma^{(+)} + \Sigma^{(-)} = \Sigma$ and eq. [[eq:normalisation]].

** Why =eigh= is the right solver (whiten-then-diagonalise)

Let $\Sigma \succ 0$ and take its Cholesky factorisation $\Sigma = LL^{\top}$. Substituting $v = L^{\top}w$ into eq. [[eq:pencil-norm]]:

\begin{align*}
\Sigma^{(+)} w &= \mu \, \Sigma w \\
\Sigma^{(+)}L^{-\top}v &= \mu\, LL^{\top}L^{-\top}v \\
L^{-1}\Sigma^{(+)}L^{-\top}v &= \mu\, v
\end{align*}

Write $A = L^{-1}\Sigma^{(+)}L^{-\top}$. Then $A^{\top} = A$, so (Spectral Theorem):

- the spectrum $\{\mu_k\}$ is *real*;
- the eigenvectors $V = [v_1 \cdots v_C]$ can be chosen *orthonormal*, $V^{\top}V = I$;
- recovering $W = L^{-\top}V$ gives

  $W^{\top}\Sigma W = V^{\top}L^{-1}LL^{\top}L^{-\top}V = V^{\top}V = I$,

  i.e. eq. [[eq:normalisation]] holds *automatically*.

$L^{-1}$ is a whitening transform for $\Sigma$; the ordinary symmetric eigenproblem for $A$ is the diagonalisation. "Whiten then diagonalise" and "solve the generalised pencil" are therefore the same computation, and =scipy.linalg.eigh(Sigma_p, Sigma)= performs it. Consequences for the code:

| property                  | =numpy.linalg.eig(inv(Sigma) @ Sigma_p)= | =scipy.linalg.eigh(Sigma_p, Sigma)= |
|---------------------------+------------------------------------------+-------------------------------------|
| dtype                     | =complex128= whenever $\Im \neq 0$       | =float64= guaranteed                |
| $W^{\top}\Sigma W = I$    | no                                       | yes                                 |
| eq. [[eq:complement]] holds   | no                                       | yes                                 |
| output order              | unspecified                              | *ascending* $\mu$                   |
| explicit inverse required | yes                                      | no                                  |

The ascending order is a silent trap: reversing the spectrum yields $\mu \mapsto 1-\mu$, which still looks like a valid CSP spectrum.

* Rank restriction

** The problem

An average reference imposes $\mathbf{1}^{\top}Z_c = 0$, hence $\operatorname{rank}\Sigma \le C - 1$ and $\Sigma \not\succ 0$: the Cholesky step above does not exist. Empirically, with $C = 64$ and the reference applied, the naive route returns eigenvalues outside $[0,1]$ (observed range $[-0.309,\, 1.522]$) and a =complex128= filter matrix — a silent failure, not an exception.

** The fix

Let $\Sigma_{\text{ref}} = \tfrac12\Sigma$ with eigendecomposition

$\Sigma_{\text{ref}} = UDU^{\top}$, $D = \operatorname{diag}(d_1 \ge \cdots \ge d_C)$,

and let $r = \operatorname{rank}\Sigma_{\text{ref}}$. Define the *restricting matrix*

#+NAME: eq:restrict
\begin{equation}
P = \big[\,u_1 \; \cdots \; u_r\,\big]^{\top} \in \mathbb{R}^{r \times C}
\end{equation}

whose rows span the principal (non-null) subspace. Solve the *restricted* pencil

#+NAME: eq:restricted-pencil
\begin{equation}
\big(P\Sigma^{(+)}P^{\top}\big)\,\tilde{w} = \mu \,\big(P\Sigma P^{\top}\big)\,\tilde{w}
\end{equation}

in which $P\Sigma P^{\top} \succ 0$ by construction, then lift back:

#+NAME: eq:lift
\begin{equation}
w = P^{\top}\tilde{w}, \qquad W = P^{\top}\tilde{W} \in \mathbb{R}^{C \times r}
\end{equation}

so =filters_= $= W^{\top} \in \mathbb{R}^{r \times C}$. This is what MNE does by default (=restr_type="restricting"=), which is why its =filters_= is $(63, 64)$ and not $(64, 64)$ on average-referenced data.

*Verification target:* eq. [[eq:restricted-pencil]] reproduces MNE's spectrum to $\sim 10^{-13}$.

* Component selection and ordering

Order the eigenpairs by $\mu$ descending, $\mu_1 \ge \cdots \ge \mu_r$. Two selection rules for $n_{\text{comp}}$ filters:

- *Symmetric extremes* (mine): take $m = \lfloor n_{\text{comp}}/2 \rfloor$ from each end, indices $\{1,\dots,m\} \cup \{r-m+1,\dots,r\}$.
- *Mutual information* (MNE, two-class case): take the $n_{\text{comp}}$ largest values of $|\mu_k - \tfrac12|$.

Both target maximally class-imbalanced filters. They coincide when the spectrum is symmetric about $\tfrac12$ and can differ otherwise, since the second rule may return an unbalanced split (e.g. three high, one low).

* Feature extraction

** Variance of a projection

For a filter $w$ and epoch $X_i$, the projected signal is $s = w^{\top}X_i \in \mathbb{R}^{1 \times T}$ and its mean power is

#+NAME: eq:power
\begin{equation}
\frac{1}{T}\lVert s \rVert_2^2
= \frac{1}{T}\, w^{\top} X_i X_i^{\top} w
\end{equation}

which is the quadratic form that eq. [[eq:objective]] is built from — the objective and the feature are the *same quantity*, one averaged over a class and one evaluated per epoch.

** Feature vector

With $F = [\,w_{k}\,]^{\top} \in \mathbb{R}^{n_{\text{comp}} \times C}$ the selected filters as rows, define for epoch $i$ and component $k$

#+NAME: eq:feature
\begin{equation}
f_{ik} = \log\!\left(\frac{1}{T}\sum_{t=1}^{T} \big[F X_i\big]_{kt}^{2}\right)
\end{equation}

In code:

#+BEGIN_SRC python
X_prj = np.matmul(W_selected, X)      # (n_epochs, n_comp, n_times)
features = (X_prj**2).mean(axis=2)    # eq. (power)
features = np.log(features)
#+END_SRC

*Mean square, not variance.* Eq. [[eq:feature]] uses $\frac1T\sum s_t^2$, not $\frac{1}{T-1}\sum (s_t - \bar{s})^2$. The two agree iff $\bar{s} = 0$, which the band-pass filter supplies — the same assumption as in [[*Zero-mean assumption][Zero-mean assumption]].

*Why the logarithm.* Two reasons:
1. Band power is approximately log-normally distributed across epochs; the log makes the features approximately Gaussian, which is the generative assumption of the LDA that consumes them.
2. It converts the scale ambiguity into an additive constant: replacing $w_k \mapsto \alpha w_k$ shifts $f_{ik}$ by $2\log|\alpha|$, a per-feature offset that LDA absorbs into its intercept.

* Identifiability

Three symmetries of the solution, each of which the verification harness must quotient out before comparing against MNE:

1. *Scale.* $J$ and $\mu$ are invariant under $w \mapsto \alpha w$. Different solvers impose different normalisations (=eig= returns $\lVert w \rVert_2 = 1$; =eigh= returns $W^{\top}\Sigma W = I$). Compare unit-normalised rows.
2. *Sign.* $w$ and $-w$ give identical features, since eq. [[eq:power]] is quadratic. Compare $|\cos|$.
3. *Rotation within a degenerate subspace.* If $\mu$ has multiplicity $>1$, only the *eigenspace* is determined; any orthonormal basis of it is a valid set of filters. Filters with $\mu \approx \tfrac12$ are therefore not individually reproducible, while the extremes ($\mu \to 0, 1$) are well separated and must match tightly.

* Note on the subject's notation

The subject writes the data as $X \in \mathbb{R}^{d \times N}$ with $d = ch \times time$, i.e. each epoch flattened into a single vector of length $C \cdot T$. CSP does *not* operate on that space: the projection acts on the channel axis only ($C$-dimensional), with time treated as the sample axis of the covariance estimate. Under the flattened reading $W$ would be $CT \times CT$ ($\approx 10^{4}$ square) and the covariance would be rank-1 per epoch. The relation $W^{\top}X = X_{CSP}$ is correct only with $X$ read as a single epoch $X_i \in \mathbb{R}^{C \times T}$ and $W \in \mathbb{R}^{C \times C}$.
